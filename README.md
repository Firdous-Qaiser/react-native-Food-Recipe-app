# React Native Food Recipe App

A modern mobile application built with **React Native** and **Expo** that allows users to browse food categories, explore recipes by category, search recipes by name, and view detailed cooking instructions. The app integrates with **TheMealDB API** and provides a smooth, responsive user experience with animations and optimized API caching.
